# 加农炮小游戏
## 简介
&emsp;&emsp;非原创，参考《Android大学教程》中文版第6章。
## 主要知识点
- 将定制的`view`与布局绑定
- 使用*raw*资源文件夹
- `Activity`和`Fragment`的生命周期方法
- 用`SoundPool`和`AudioManager`添加声音
- 用`Thread`、`SurfaceView`和`SurfaceHolder`实现逐帧动画
- 简单的冲突检测
- 用`Paint`和`Canvas`绘制图形
